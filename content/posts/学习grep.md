```bash
$ grep [-acinv] [--color=auto] 'string' filename
```

|     参数     |             注释             |
| :----------: | :--------------------------: |
|      -a      | 将二进制文件以字符串形式搜索 |
|      -c      |  计算搜索到目标字符串的次数  |
|      -i      |          无视大小写          |
|      -n      |           输出行号           |
|      -v      |   搜索不带目标字符串的结果   |
| --color=auto |    搜索到的关键字添加颜色    |
